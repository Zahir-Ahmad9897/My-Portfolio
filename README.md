## 🚀 About

A **production-grade personal portfolio** built with React 18 + Vite, showcasing my work as an **AI & Computer Vision Engineer** based in Peshawar, Pakistan. The site is modular, responsive, and performance-optimized.

> *"Building intelligent systems that see, understand, and act."*

---

## ✨ Features

- ⚡ **Blazing fast** — powered by Vite 6 with instant HMR
- 🎨 **Modern UI** — glassmorphism, gradients & micro-animations
- 📱 **Fully Responsive** — mobile-first layout across all devices
- 🧩 **Modular Architecture** — clean component-based structure
- 🤖 **AI Chat Assistant** — built-in portfolio chatbot
- 📊 **Live GitHub Projects** — real project data with live repo links
- 📬 **Contact Form** — integrated email contact section
- 🌑 **Dark Mode Design** — sleek dark aesthetic throughout

## 🏃 Getting Started

### Prerequisites
- Node.js `>= 18`
- npm `>= 9`

### Installation

```bash
# Clone the repository
git clone https://github.com/Zahir-Ahmad9897/My-Portfolio.git

# Navigate into the project
cd My-Portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```

Output will be in the `dist/` folder.
